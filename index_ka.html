შესავალი

JavaScript(იკითხება: ჯავასკრიპტი; აბრევ. JS) — პროგრამირების ერთ-ერთი ფართოდ გავრცელებული ენაა და ერთადერთი პროგრამირების ენა, რომელიც გამოიყენება ვებ დეველოპმენტში. სახელწოდება JavaScript, როგორც სავაჭრო ნიშანი დარეგისტრირებულია ფირმა Sun Microsystems, Inc-ის მიერ. JavaScript შექმნილია კომპანია Netscape-ის მიერ, და პირველად ჩართული იყო Netscape Navigator ბრაუზერში. მისი თავდაპირველი სახელწოდება იყო Mocha, შემდეგ LiveScript. მაგრამ იმ დროისთვის, პოპულარულობის პიკში იყო Java და მარკეტოლოგებმა გადაწყვიტეს, რომ "Java"-ს მსგავსი დასახელება უფრო პოპულარულს გახდიდა ენას. ასე წარმოიშვა სახელი JavaScript. ასევე Javascript-ი Java-ს "შვილობილი" ენა უნდა ყოფილიყო, მაგრამ შემდგომში, JavaScript სხვა გზით განვითარდა, მეტად დაიხვეწა და ჩამოყალიბდა დამოუკიდებელ ენად, რომელსაც გააჩნია თავისი სპეციფიკაცია - ECMAScript..

სხვადასხვა ენაზე დაწერილი პროგრამის შესრულების ორი ვარიანტი არსებობს - კომპილაცია და ინტერპრეტაცია:

  • კომპილაცია - არის პროცესი, როდესაც პროგრამის კოდი, სხვა პროგრამის სპეციალური ინსტრუმენტის - კომპილატორის დახმარებით გადაიწერება პროცესორისათვის გასაგებ ენაში - სამანქანო ენაში და ამის შემდეგ ხდება მისი შესრულება.
  • ინტერპრეტაციაარის პროცესი, როდესაც პროგრამის კოდი გადაეცემა ინტერპრეტატორს , რომელიც კოდში მითითებულ ინსტრუქციებს ასრულებს კოდის სამანქანო ენაზე გადათარმნის გარეშე.
რა ცოდნაა საჭირო?

JavaScript-ის შესწავლამდე აუცილებელია საბაზისო ცოდნა შემდეგ მოთხოვნებზე:

  • ინტენრეტისა და ქსელის(WWW) მუშაობის ზოგადი პრინციპების ცოდნა.
  • HTML/CSS-ის კარგი ცოდნა, ვინაიდან JavaScirpt-ი უშუალოდ HTML DOM-ზე მოქმედებს.
  • პროგრამირების საფუძველბისა და პრინციპების ცოდნა
Your First Program

"Your first program will be perfect! No errors, no bugs, ..."Hello, World!"

Hello world — უმარტივესი პროგრამაა, რომელიც ამა თუ იმ საშუალებით გამოუტანს მომხმარებელს შეტყობინებას „Hello, World!“. ამ პროგრამის დაწერის ტრადიცია მოდის დენის რიტჩის და ბრაიან კერნიგენის კლასიკური ნაშრომიდან „The C Programming Language“. დღეს უმეტესობა ელემენტარული პროგრამირების ენის სახელმძღვანელოსი იწყება ამ პროგრამით, როგორც უმარტივესი მაგალითია ყველა შესაძლებლიდან.

"Hello World" JS იმპლემენტაცია:

console.log("Hello, World!");
ცვლადები (Variables)

ცვლადები წარმოადგენს რაიმე ტიპის მონაცემების შემნახველს. ჯავასკრიპტში არსებობს სამი ცვლადის სახეობა:

  • var
  • let
  • const

ცვლადის სახელებს ეწოდებათ იდენტიფიკატორები. ცვლადების სახელდებისას აუცილებლად უნდა იქნეს გათვალისწინებული შემდეგი მოთხოვნები:

  • ცვლადის სახელის პირველი სიმბოლო აუცილებლად უნდა იყოს ლათინური ანბანის დიდი ან პატარა ასო, ან ქვედა ტირე ან დოლარის ნიშანი;
  • ცვლადის სახელში დაუშვებელია სასვენი ნიშნებისა და გამოტოვებული ადგილების გამოყენება;
  • ცვლადის სახელი მგრძნობიარეა სიმბოლოთა რეგისტრის მიმართ. შესაბამისად "name" და "Name" სხვადასხვა ცვლადებია
  • ცვლადის სახელში სიმბოლოთა რაოდენობა არ არის შეზღუდული.
  • ჯავასკრირპტის რეზერვირებული სიტყვების ცვლადებად ვერ გამოვიყენებთ. (მაგალითად, როგორებიცაა: for, while, var, etc).

სინტაქსი:

//Syntax
var variableName = value;
var x = 10;
let name = 'Jon Snow'
გლობალური და ლოკალური ცვლადები

როდესაც, ჩვენ ცვლადს ვქმნით ფუნქციის გარეთ, მას ეწოდაბა გლობალური ცვლადი, რადგან იგი ხელმისაწვდომი ხდება ნებისმიერი სხვა ფუნქციისთვის, და მთლიანად პროგრამის ნებისმიერი ნაწილისთვის. ხოლო თუ ცვლადს ვქმნით ფუნქციის შიგნით, ის არის ლოკალური ცვლადი, რადგან მისი გამოყენება შესაძლებელია მხოლოდ ამ ფუნქციაში. სხვაგვარად რომ ვთქვათ იგი "ჩაკეტილია" ფუნქციაში და პროგრამის გარეთ ის არ არსებობს. ჯავასკრიპტის გლობალურ ცვლადს წარმოადგენს var. ხოლო ლოკალური ცვლადებია let და const, რომლებიც წარმოდგენილი იქნა ECMASCript2015-ში.

უკეთესი ვიზუალიციისთვის განვიხილოთ კოდი:

if (true) { var x = 5; } console.log(x); // 5 >

ცოლო let-ის გამოყენების შემთხვევაში გვექნება შემდეგი:

if (true) { let y = 5; } console.log(y); // ReferenceError: y is not defined
მონაცემთა ტიპები

ჯავასკრიპტში გვხვდება სულ 7 სახის მონაცემთა ტიპი, აქედან პირველ ექვს ეწოდება primitives (პირველადები) ხოლო მეშვიდე არის ობიექტები, რომელიც ამათ შორის ყველაზე ეფექტური და კომპლექსურია:

  • Numbers - ნებისმიერი სახის რიცხვები (მაგ: 17.45,-45, 1E+12...)
  • Strings - სტრიქონული ტიპი - ტექსტი და სიმბოლოები (მაგ: "hello", "a",...)
  • Boolean - ლოგიკური ტიპის მნიშვნელობები true და false
  • Symbol - (ECMAScript 2015). მონაცემთა ტიპი, რომელიც იღებს უნიკალურ, შეუცვლელ მნიშვნელობას.
  • Objects (ობიექტები) - ობიექტი არის კოდის ფრაგმენტი რომელიც მოიცავს მეთოდებსა და თვისებებს, თვისება არის ობიექტის მახასიათებელი, ცვლადს რომელიც აღწერილია ობიექტში ეწოდება ობიექტის თვისება, ხოლო ფუნქციას რომელიც აღწერილია ობიექტში - ამ ობიექტის მეთოდი . ჯავასკრიპტში ობიექტი აღიქმევა რეალურ ცხოვრებაში არსებულ ობიექტისავით, მაგალითად მანქანა, ჩვენ შეგვიძლია მას მივცეთ მისთვის უნიკალური ფერი, ნომერი, ტიპი და ა.შ. ობიექტში ასევე შედის მასივები (Arrays).მეტი ობიექტებზე =>
  • undefined - განუსაზღვრელი მნიშვნელობის მქონე მონაცემი, მაგ: ცვლადს, რომელსაც არ აქვს მნიშვნელობა მინიჭებული
  • null - ამ მნიშვნელობას არანაირი კავშირი არ აქვს ზემოთ ჩამოთვლილ ტიპებთან, ეს არის ცალკე ტიპის ერთადერთი შესაძლო მნიშვნელობა, ანუ ეს არის სპეციალური მნიშვნელობა რომელიც პირდაპირი მნიშვნელობით გულისხმობს "არაფერს".

Although these data types are a relatively small amount, they enable you to perform useful functions with your applications. First six data types are called primitives. Objects and functions are the other fundamental elements in the language.

პირობითი წინადადებები

პირობიტი წინადადებები გამოხატავს რაიმე დამოკიდებულებას, რომლიც ემქვედებარება გარკვეულ ლოგიკას. ჯავასკრიპტში ამისთვის გამოიყენება if, else, else if და switch-case ოპერატორები. მათ იყენებენ, რათა შევამოწმოთ კონკრეტული პირობის მცდარობა ან სამართლიანობა და შესაბამისად ვიმოქმედოთ პასუხის შემთხვევაში.

if გამოიყენება, რათა შევასრულოთ რაიმე მოქმედება, თუკი პირობა არის სამართლიანი (true). მასთან ერთად ხშირად იყენებ else წინადადებას, რათა პირობის მცდარობის (false) შემთხვევაში შესრულდება ის მოქმედება რაც else-ის კოდშია მითითებული. სინტაქსი შემდეგნაირია:

if (condition) { //მოქმედება_თუ_პირობა_სამართლიანია; } else { //მოქმედება_თუ_პირობა_მცდარია; }

else if წინადადებას იგივე პრინციპი აქვს, განსხვავებით, რომ შეგვიძლია მისი მრავალჯერ გამოყენება სხვადასხვა პირობების დასატესტად:

if (condition_1) { statement_1; } else if (condition_2) { statement_2; } else if (condition_n) { statement_n; } else { statement_last; }

სიტყვა გასაღები switch კითხულობს ცვლადის მნიშვნელობას, სიტყვა გასაღები case საშუალებას იძლევა განიხილოს, ჩამოითვალოს ცვლადის სავარაუდო მნიშვნელობები და აგრეთვე შესაბამისი მოქმედებები. break კი უზრულვეფოფს ციკლის დამთავრებას იმ შემთხვევაში თუ მის მშობელ case - ში მითითებული პირობა შესრულდება. default არის ალტერნატიული ქწმედება თუ არც ერთ case - ში მითითებული პირობა არ სრულდება.

სინტაქსი:

switch (expression) { case value1: //თუკი expression დაემთხვა value1-ს შესრულდება ამ case-ში მოცემული კოდი break; case value2: //თუკი expression დაემთხვა value2-ს შესრულდება ამ case-ში მოცემული კოდი break; ... case valueN: //თუკი expression დაემთხვა valueN-ს შესრულდება ამ case-ში მოცემული კოდი break; [default: //optional //თუკი არცერთი პირობა არ შესრულდა [break;]] }
ციკლური ოპერაციები

ამა თუ იმ ამოცანის გადასაჭრელად ხშირად საჭირო ხდება ერთი და იგივე გამოთვლების მრავალჯერ ჩატარება. განმეორებად ბრძანებათა ჯგუფს ეწოდება ციკლი. javascript - ში ციკლები იწერება შემდეგი ოპერატორების დახმარებით : while და for .

While ოპერატორი

while ციკლი არის ასე ვთქვათ წინაპირობითი, პრეპირობითი ანუ ციკლში პირობის შემოწმება/შესრულება ხდება ციკლის გამოძახებამდე , ანუ ციკლის ტანის შესრულებამდე გამოითვლება ლოგიკური პირობის მნიშვნელობა, თუ პირობა ჭერშმარიტია მაშინ ციკლიც სრულდება, ამის შემდეგ ისევ მოწმდება პირობა, ციკლი მეორდება მანამ სანამ პირობა არის true.

while (condition_is_true) { statement //do this }
For ოპერატორი

ციკლის ეს ოპერატორი თავი თავში მოიცავს ცვლადს, ე.წ ციკლის პარამეტრს ან ციკლის მთვლელს. ციკლი იწყება სიტყვა გასარებით for შემდეგ ფრჩხილებში იწერება კონსტრუქცია, რომელიც მოიცავს ციკლის პარამეტრის ინიციალიზაციას, ლოგიკურ გამოსახულებას და ამ პარამეტრის მოდიფიკაციის ანუ ცვლილების ჩანაწერს

Syntax:

for ([initialExpression]; [conditionExpression]; [incrementExpression]) statement

ჯავასკირპტს ასევე გააჩნია სხვა მეტად თანამედროვე ციკლის ოპერატორები როგორებიცაა: do..while, for/in და for/of. ინფორმაცია აქ

ფუნქციები

ფუნქცია ეს არის პროგრამის სახელდებული ფრაგმენტი, სხვადასხვა ინსტრუქციების ერთობლიობა, რომლის განსაზღვრაც ხდება ერთხელ მაგრამ გამოძახება შეიძლება მრავალჯერ.

ფუნქციის შექმნა ხდება შემდეგნაირად: სიტყვა გასაღებიfunction-ს მიყოლებული:

  • ფუნქციის სახელი (სურვილისამებრ)
  • პარამეტრები, გამოყოფილი მძიმით (ასევე შესაძლოა ფუნქციას არ ქონდეს პარამეტრები).
  • და კოდი, რომილიც მოთავსებული იქნება ფიგურილ ბრჭყალებში: {...}.

სინტაქსი:

function functionName(parameter1, parameter2, ... parameterN) { code block }; //მაგალითი (რიცხვის კვადრატის დამთვლელი ფუნქცია): function square(number) { return number * number; }
